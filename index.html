Literature & Instructions

Thermistor Readings:
Hold MODE + NEXT → TST → DOWN to CP → ENTER → CP ON → Enter DT, ST, ET, AT, Current.

Element Readings:
MODE + NEXT → TST → UP to UE → Enter UT → ENTER twice → Enter Current.
Repeat for LE + LT.

Normal Temps:
DT < 160°F (rises 10–15°F each test)
ET < 70°F
ST = 38–90°F

Capacitor Test:
MODE + NEXT → TST → DOWN to CP → CP ON → Test VAC across capacitor.

EXV Ohm Values:
OY 87.6, RB 88.2, GB 44.7, GY 47.5, GR 43.9, GO 44.3
